# ShengHang

## 使用方法

### 1. 启动后端服务

首先需要启动Django后端服务：

```bash
python manage.py runserver
```

### 2. 启动前端服务

启动前端服务：

```bash
python -m http.server 3000
```

### 3. 访问前端页面

```bash
访问 http://127.0.0.1:3000
```
